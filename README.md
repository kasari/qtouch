# qtouch
qtouch is a command to create an empty file quickly

## example
```sh
% tree                                                                                                                                                                                             [master]
.
├── LICENSE
├── README.md
└── main.go

% qtouch hoge/fuga/poyo
% tree                                                                                                                                                                                             [master]
.
├── LICENSE
├── README.md
├── hoge
│   └── fuga
│       └── poyo
└── main.go
```
